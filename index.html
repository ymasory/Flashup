Studying. In plain text.

A simple markup language for creating flashcards

Overview

Every programmer knows the fastest way to do anything is in a text editor. Flashup takes that philosophy to flashcards, providing a markup system in the spirit of Markdown and reStructured Text. A PDF traslator is provided for easy printing. A translator to Anki's format (experimental!) is provided for flashcard review on Linux, Mac, Windows, iPhone, and Android. More translators on the way to target popular Android and iPhone flashcard apps. Flashup provides simple text formatting and special modes for writing program code and LaTeX.
